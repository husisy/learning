# first nextjs app

```bash
npm install react react-dom next #which will create package.json and package-lock.json in the current directory
# manually add "scripts": { "dev": "next dev" } to package.json
# an example package.json can be found below (don't copy it directly)

npm run dev
```

`package.json`

```json
{
  "scripts": {
    "dev": "next dev"
  },
  "dependencies": {
    "next": "^13.4.19",
    "react": "^18.2.0",
    "react-dom": "^18.2.0"
  }
}
```
